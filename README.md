# Resolvendo Códigos em Python com GitHub Copilot

Repositório desenvolvido durante a Formação GitHub Certification da DIO.

## Sobre o projeto

Este repositório reúne soluções para desafios de lógica de programação em Python, utilizando o GitHub Copilot como ferramenta de apoio ao desenvolvimento.

O objetivo é praticar conceitos fundamentais da linguagem Python, como manipulação de strings, estruturas condicionais, operadores matemáticos e entrada de dados, além de explorar recursos de inteligência artificial aplicados à programação.
